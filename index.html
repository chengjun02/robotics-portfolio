Projects

EE4308 Project 1 (Jan – Mar 2025)

Developed a mobile robot system using a regulated Pure Pursuit controller and A* path planner with Savitzky-Golay smoothing. Implemented in simulation and tested on a physical TurtleBot navigating obstacle-rich maps using ROS2, RViz, and Gazebo.

EE4308 Project 2 (Mar – May 2025)

Simulated aerial drone behavior in ROS2. Implemented behavior logic for different flight stages (takeoff, cruise, landing), a PID-based controller for flight speed, and an Extended Kalman Filter-based estimator using sensor fusion to estimate drone position. Tools used: ROS2 and RViz.

EE4705 (Aug – Nov 2025)

Designed an arm trajectory planner for a Kinova robotic arm using the STOMP algorithm. Implemented using MATLAB and Simulink for motion planning and optimization.

ME2104 – Teaching Assistant

Assisted weekly electronics labs where students explored fundamental electrical components like motors, transistors, and LEDs. Guided students toward building a robotic car for their final project.
